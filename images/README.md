# Basai - Image Assets Directory

This directory contains all image assets for the Basai student accommodation website.

## Directory Structure

- **hostels/**: Images of student accommodation properties
  - hostel1.jpg through hostel10.jpg - Main property images
  - detail/*.jpg - Detailed room and facility images
  
- **avatars/**: User profile pictures and student testimonials
  - user1.jpg through user10.jpg - Student profile pictures

- **team/**: Team member photos for About page
  - member1.jpg through member4.jpg - Team member portraits

- **partners/**: Partner organization logos
  - partner1.png through partner6.png - Logo images

- **payments/**: Payment method logos and icons
  - esewa-logo.png - eSewa payment logo
  - khalti-logo.png - Khalti payment logo

- **about/**: About page specific images
  - team.jpg - Team group photo
  - office.jpg - Office/workspace images

- **reviews/**: User-uploaded review images
  - review-*.jpg - Guest room and facility photos

## Image Specifications

### Hostel/Property Images
- **Dimensions**: 1200x800px (3:2 aspect ratio)
- **Format**: JPG, optimized for web
- **Size**: < 200KB per image

### Avatar Images
- **Dimensions**: 200x200px (1:1 aspect ratio)
- **Format**: JPG or PNG
- **Size**: < 50KB per image

### Logo Images
- **Dimensions**: Variable, maintain aspect ratio
- **Format**: PNG with transparency
- **Size**: < 30KB per image

### Team Photos
- **Dimensions**: 400x400px (1:1 aspect ratio)
- **Format**: JPG
- **Size**: < 100KB per image

## Placeholder Images

For development/presentation purposes, you can use:
- https://via.placeholder.com/1200x800 for hostel images
- https://via.placeholder.com/200x200 for avatars
- https://via.placeholder.com/400x400 for team photos
- https://via.placeholder.com/300x100 for partner logos

Or use free stock photo services:
- Unsplash (https://unsplash.com/) - High-quality free images
- Pexels (https://pexels.com/) - Free stock photos
- Pixabay (https://pixabay.com/) - Free images and videos

## Notes

- All images should be optimized for web before uploading
- Use descriptive filenames (e.g., thamel-student-house-exterior.jpg)
- Include alt text for accessibility
- Consider using responsive images with srcset for better performance
